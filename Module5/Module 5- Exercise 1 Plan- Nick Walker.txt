Decide how many categories will be included in the game.

Decide how many questions will be in each category.

Create each question and phrase each answer in the form of a question (Jeopardy style).

Assign a point value to each question.

Initialize the player’s score to zero.

Display the game board showing categories and available question values.

Allow the player to choose a category and a question value.

Retrieve and display the selected question.

Allow the player time to think and enter their answer.

Check whether the player’s answer is correct.

If the answer is correct, add the question’s point value to the score.

If the answer is incorrect, subtract the question’s point value from the score.

Mark the question as “used” so it cannot be selected again.

Return to the game board and show remaining available questions.

Repeat the question‑selection process until all questions have been used or the game ends.

Display the player’s final score.
Winning Player has the highest score.

Game END.
